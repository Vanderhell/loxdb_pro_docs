# Contributing

This repository accepts documentation improvements only.

## Allowed changes

- clarity improvements in existing docs
- typo fixes
- API contract clarifications
- release-note and roadmap updates

## Not allowed here

- source code contributions
- non-public runbooks or private infrastructure details
- secrets, personal data, or local machine paths

## Pull request checklist

1. Scope is documentation-only.
2. No local absolute paths.
3. No personal/private contact data.
4. Technical claims are consistent with current public docs.
