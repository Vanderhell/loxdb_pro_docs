# Commercial Access

LoxDB PRO is a proprietary commercial extension layer for LoxDB.

This repository contains only the public technical overview. It does not contain the PRO source code, private headers, binaries, validation logs, or commercial integration packages.

---

## What commercial access may include

Depending on the agreed scope, commercial access can include:

- PRO source code access,
- binary/static library package,
- integration documentation,
- validation logs,
- compiler/toolchain build evidence,
- support for selected modules,
- license terms for production deployment,
- integration guidance for storage, security, CLI, and operational workflows.

---

## Evaluation package

A commercial evaluation package should define:

1. target platform,
2. compiler/toolchain,
3. required modules,
4. storage backend,
5. security requirements,
6. validation evidence required by the evaluator,
7. permitted evaluation duration,
8. redistribution and deployment limits.

---

## Licensing boundary

The open-source `loxdb` core and LoxDB PRO are separate licensing surfaces.

- `loxdb` core: governed by the license of the public core repository.
- `loxdb_pro`: proprietary commercial software unless a separate written agreement says otherwise.

This repository does not grant any production license for LoxDB PRO.
